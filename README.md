# Модульний контроль №2 (Варіант 3) — Практичні завдання з JavaScript

Цей репозиторій містить комплексне рішення практичних завдань з курсу JavaScript, об'єднаних в один спільний робочий файл. Код структурований, задокументований та готовий до автоматичної перевірки.

## 📂 Структура проєкту

* **`Task1.js`** — єдиний головний скрипт, який містить логіку для всіх чотирьох завдань модульного контролю:
  1. **Завдання 1 (Масиви та замикання):** Робота з методами перебору масивів (HOF) та збереженням внутрішнього стану.
  2. **Завдання 2 (Об'єктно-орієнтоване програмування):** Реалізація класів, наслідування, інкапсуляції та методів об'єктів.
  3. **Завдання 3 (Асинхронний JavaScript):** Робота з `Promise`, обробка станів та асинхронних запитів.
  4. **Завдання 4 (Робота з API):** Імітація або інтеграція з асинхронними джерелами даних (Async/Await).
* **`README.md`** — документація та інструкція користувача.

## 🚀 Як запустити проєкт локально

Переконайтеся, що на вашому комп'ютері встановлено середовище виконання **Node.js**.

1. Відкрийте термінал або PowerShell у папці проєкту (`C:\Навчання`).
2. Виконайте команду для запуску файлу:

```bash
node Task1.js
